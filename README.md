## Usefull Terminal Commands

**Git Commands**

- `git switch -c <branch-name>` creates a branch and checkout 
- `git reset --soft main` deletes all your individual commits but keeps all your code changes staged and ready to go
    - `git commit -m "Your new single commit message"`
    - `git push --force-with-lease`
